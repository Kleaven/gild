# Gild persona art — generation kit

Drop 8 PNG files here with these EXACT names and the crowd + auth panels light
up automatically (no code changes needed):

  potter.png  guitarist.png  photographer.png  gardener.png
  lifter.png  painter.png    baker.png         dj.png

## Specs (important for a clean result)
- **Transparent background PNG** (preferred) so they sit on any surface.
  If your tool can't do transparent, a pure-white background also works.
- **Square-ish, ~800×1000px**, head-and-shoulders, centred, SAME zoom/framing
  in every image so the crowd lines up.
- **Black ink line-art, no colour.**

## Tool
Best: **Recraft** (recraft.ai) — set Style = "Hand-drawn"/"Line art", turn ON
transparent background, generate one at a time so the style stays consistent.
Also good: Ideogram, Midjourney, DALL·E (ChatGPT). Generate ONE per prompt.
